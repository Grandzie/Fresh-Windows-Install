How to do a Fresh Windows Install


Data warning: Deleting partitions erases all data on that drive. Back up first.

 

Check your Local Disk C size

Open File Explorer, This PC and look at Local Disk (C:) to see its’ size. Remember this number for the installer.

 

Boot from the USB in Bios

Plug the Windows USB into a USB port in the back of the PC. Restart PC and tap DEL on your keyboard to access BIOS. Press F7 on your keyboard to go to advanced mode, head over to the Boot tab and set Boot Option #1 to the USB stick. Save changes and reboot.

 

Follow the Windows installer prompts. When you encounter the screenshots shown in the document below, follow them exactly.


Partition Guide

Identify the correct disk by size (the one that matches your C: drive).

Delete all four partitions on that disk (System, MSR, Primary, Recovery) until only Unallocated Space remains for that disk. Do NOT delete partitions on other disks.


Select “Unallocated Space” on the disk that matches your C: drive’s size. Click Next.

Do NOT restore from backup
When this screen appears, click More options → Set up as new PC.
